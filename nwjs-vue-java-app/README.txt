========== Build using NodejS ======================

0. Required:

- nodejs (+ npm)

1. Install nwjs-builder-phoenix
$ npm install

2. Run
$ npm run start

3. Build
$ npm run dist

========== Build using Gradle ======================

0. Required:

- Gradle

1. Run
$ gradle run

2. Build
$ gradle build
